# Do Different

A frontend-only MVP for discovering curated experiences based on a user's current mood and intent.

## What it does

- Guides users through a short discovery flow
- Recommends curated experiences
- Learns locally from Love it / Not for me / Tell me more interactions
- Opens provider pages for booking outside the app

## What it does not do yet

- No backend
- No authentication
- No payments
- No in-app booking
- No AI recommendation engine
- No provider portal

## Tech stack

- React
- Vite
- CSS
- localStorage / sessionStorage

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
